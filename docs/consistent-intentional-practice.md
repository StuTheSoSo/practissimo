# Consistent, Intentional Practice

## Mission

The core mission of PracticeQuest should be:

- Consistent practice
- Intentional practice
- Purposeful progress

This means the app should feel less like a generic habit game and more like a focused practice coach: guiding each session, reinforcing a daily habit, and helping users practice with intention rather than only logging time.

## Recommended adjustments

### 1. Messaging

- Add a mission statement to onboarding and home screens:
  - "Consistent. Intentional. Practice."
  - "Build a daily habit with focused, purposeful sessions."
- Make the home hero more mission-driven:
  - "Start a short, intentional practice session" instead of only "Start Practice."
  - Keep streaks and weekly target visible, but secondary to the mission.

### 2. Practice flow

- Ask users to define a clear session focus before starting:
  - "What are you practicing today?"
  - "Select the skill area for this session."
- Use categories as practice purpose, not just labels.

### 3. Education

- Add help content explaining intentional practice and consistency.
- Emphasize:
  - one or two goals per session
  - short, regular practice over infrequent long sessions
  - using session notes or reflections to stay deliberate

### 4. Support features

- Add guided session templates / practice plans:
  - "Warmup + Technique"
  - "Repertoire polish"
  - "Scale + review"
- Add a small home card for practice focus or recommended next exercise.
- Use reminders to protect habit and encourage the next intentional session.

### 5. De-emphasize

- Avoid making XP/streaks the main story.
- Keep quick-log available, but not the primary path.
- Avoid over-gamifying the core practice flow.

## What should be added

- A visible mission statement in onboarding and home copy.
- A session focus prompt in `practice.page.ts`.
- A help/FAQ entry explaining why consistent, intentional practice matters.
- A practice plan/template chooser.
- A mission-focused home preview or recommended focus card.
